About

Inspired by pngtocss, Gradient Finder is an entirely in browser way to convert an image to it's CSS gradient.

Currently supports only linear gradients.

Methods:

GradientFinder {
    fromUrl: function(url, onload),
    fromCanvas: function(canvas)
};

Returns:

Gradient {
    stops: [ array ],
    angle: integer,
    toCss: function() // returns String
};
